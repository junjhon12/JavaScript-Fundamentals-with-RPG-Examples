# RPG JavaScript Projects

This directory contains a collection of JavaScript files, each representing a small project or coding exercise with an RPG theme.

## Projects

*   **00-rpgCharCard.js**: A script to create a character card for an RPG.
*   **01-merchantShop.js**: Simulates a merchant's shop with items to buy and sell.
*   **02-inventoryManager.js**: A script to manage a player's inventory.
*   **03-dungeonGatekeeper.js**: A gatekeeper script that might pose a riddle or require a key.
*   **04-combactSimulator.js**: A script to simulate a combat encounter.
*   **05-diceRoller.js**: A utility to roll different types of dice.
*   **06-characterSheet.js**: A script to manage a full character sheet.
*   **07-lootFilter.js**: A script to filter loot based on certain criteria.
*   **08-dataFetcher.js**: A script that might fetch game data from an external source.
*   **09-counterApp.js**: A simple counter application.
*   **10-songPlayList.js**: A script to manage a playlist of songs, perhaps for game ambiance.
*   **11-unitFactory.js**: A script to create different types of game units.
*   **12-mageClass.js**: A script defining a Mage class with specific spells and abilities.
*   **13-healthGuard.js**: A script to protect or manage a character's health.
*   **14-gameManager.js**: A script to manage the overall game state.
*   **15-skillTreeSystem.js**: A script to manage a character's skill tree.
*   **16-classyTasks.js**: A script that likely involves Object-Oriented programming with classes.
